# Skills Overview

Skills are reusable modules that provide coding standards or specialized functionality.

## Skill Types

| Type | How it loads | Example |
|------|--------------|---------|
| **Context** | Auto-loads based on tech stack | Coding standards |
| **Command** | User invokes with /skill-name | Presentation creator |

## Coding Standards

Auto-load based on your project's tech stack:

- [Python](/features/skills/standards-python) - PEP 8, type hints, async patterns
- [TypeScript](/features/skills/standards-typescript) - Strict types, Zod, utility types
- [JavaScript](/features/skills/standards-javascript) - ES Modules, async/await, Express
- [Shell/Bash](/features/skills/standards-shell) - Defensive scripting, ShellCheck

## Tool Skills

Invoked manually or when relevant:

- [YouTube Transcript](/features/skills/youtube-transcript) - Download transcripts with frame extraction
- [Slidev Presentations](/features/skills/create-slidev-presentation) - Create slide decks
- [Skill Creator](/features/skills/skill-creator) - Build your own skills

## Custom Skills

Your custom skills in `~/.claude/custom/skills/` override built-in ones with the same name.

See [Creating Skills](/guides/creating-skills) for how to build your own.
